
# Focus & Productivity Apps

This project is a web application for discovering, exploring, and managing productivity and focus apps. It provides detailed information about each app, including descriptions, ratings, reviews, download counts, sizes, and company names. Users can browse all applications, view individual app details, and get installation guidance.

![Hero App Screenshot](https://i.ibb.co.com/v4p7LgZP/Hero-app.png)



## üìù About This Project 

This project is a React-based web application that showcases a variety of productivity, focus, and task management apps. It helps users explore detailed information about each app, such as its description, ratings, reviews, downloads, and size. The platform is designed to be simple, clean, and easy to navigate, providing a smooth experience on both desktop and mobile devices.

It is perfect for students, professionals, or anyone looking to stay organized and productive. By giving clear and detailed information about apps, this project makes it easy for users to choose the right tool to focus, manage tasks, and boost productivity.

## üîç Project Overview

The project is a user-friendly web app built with **React**, utilizing **lazy loading** and **Suspense** for better performance. It fetches data from a **JSON file** containing app information and dynamically displays it in a **clean and responsive interface**.
